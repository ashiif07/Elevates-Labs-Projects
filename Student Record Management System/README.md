# Student Record Management System

This is a simple Java console program to manage student records.

---

## Features

- Add Student  
- View Students  
- Update Student  
- Delete Student  
- Exit  

---

## Usage

Menu will show like this:

1. Add Student  
2. View Students  
3. Update Student  
4. Delete Student  
5. Exit  

Enter your choice and follow instructions.

---

## Notes

- Data is not saved permanently  
- All data will be lost after closing the program
